# 4) Wireframes (low fidelity)

These are deliberately low-fi (layout + hierarchy) to show structure without “beautification”.

## Homepage

```
+----------------------------------------------------+
| Passport Services        [Log in]                  |
+----------------------------------------------------+
| H1: Passport application, redesigned               |
| Short explanation + reassurance                    |
| [Log in to start]   [Try OTP login]                |
|                                                    |
| 3 cards: What you’ll do | Why easier | After submit |
+----------------------------------------------------+
```

## Start application

```
+----------------------------------------------+
| Title: Before you start                      |
| - Time needed card                           |
| - Documents needed card                      |
| - Steps list                                 |
| City input                                   |
|                           [Start application] |
+----------------------------------------------+
```

## Form page (step-by-step)

```
+----------------------------------------------+
| Title: Application form                      |
| Stepper: Personal > Identity > Service ...   |
| Autosave badge: Saving… / Saved at …         |
|                                              |
| [Step fields]                                |
| Errors near fields                           |
| [Back]                          [Next]       |
+----------------------------------------------+
```

## Document upload

```
+----------------------------------------------+

```
